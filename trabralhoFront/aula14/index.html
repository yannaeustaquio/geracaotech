=========================== Questões ========================== 1. Questão 1: Escreva um código que verifique se um número é positivo. Caso contrário, mostre que ele é negativo. 2. Questão 2: Crie uma condição para verificar se a variável x é maior que 10. Se for, exiba "Maior que 10", senão exiba "Menor ou igual a 10". 3. Questão 3: Use if e else para verificar se uma variável idade é maior ou igual a 18. Exiba "Pode votar" se for, senão exiba "Não pode votar". 4. Questão 4: Verifique se um número é par ou ímpar usando o operador %. resposta: se(o resto da divisão do numero por 2 for zero -> é par, se não -> ímpar) 5. Questão 5: Escreva um código que verifica se uma pessoa pode dirigir, baseado na idade e no fato de possuir ou não carteira de motorista (duas condições). 6. Questão 6: Crie uma condição que mostre "Aprovado", "Recuperação" ou "Reprovado" com base em uma variável nota, considerando que >= 7 é aprovado, entre 5 e 6.9 é recuperação, e < 5 é reprovado. 7. Questão 7: Escreva um código que verifica se um número é divisível por 5 e por 3 ao mesmo tempo. 8. Questão 8: Faça um programa que exiba "Bom dia", "Boa tarde" ou "Boa noite" dependendo da hora (suponha valores entre 0 e 23). 9. Questão 9: Verifique se a soma de dois números a e b é maior que 10. Se for, exiba "Soma maior que 10", senão exiba "Soma menor ou igual a 10".
